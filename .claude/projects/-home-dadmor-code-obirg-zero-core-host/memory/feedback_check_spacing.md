---
name: Sprawdzaj podstawy wizualne
description: Przy pisaniu UI zawsze sprawdzaj odstępy, marginesy i proporcje — nie czekaj na feedback
type: feedback
---

Przy tworzeniu lub modyfikacji komponentów UI sprawdzaj spacing/marginesy między elementami. Użytkownik oczekuje, że takie podstawy jak odstęp między tytułem a listą będą poprawne od razu.

**Why:** Użytkownik zwrócił uwagę, że nie wyłapałem zbyt małego odstępu między nagłówkiem a timeline — to podstawa UI.

**How to apply:** Przy każdej zmianie UI przeglądaj wynik pod kątem spacing, proporcji i czytelności. Nie zostawiaj domyślnych/zerowych marginesów między sekcjami.
